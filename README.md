# 🚗 Car Insurance Risk Prediction

Machine Learning project focused on predicting vehicle insurance risk levels using supervised and unsupervised learning techniques. The goal was to build accurate classification models and identify meaningful risk-based segments from structured vehicle data.

---

## 📌 Project Objectives

* Predict insurance risk categories using classification models
* Compare model performance and optimize hyperparameters
* Reduce dimensionality while preserving key information
* Segment vehicles into risk-based clusters

---

## 📊 Workflow

### 1️⃣ Data Preprocessing

* Data cleaning and handling missing values
* Outlier treatment using IQR capping
* Feature selection and scaling

### 2️⃣ Exploratory Data Analysis (EDA)

* Correlation analysis
* Risk distribution analysis
* Feature importance exploration

### 3️⃣ Supervised Learning Models

* Logistic Regression
* Decision Tree
* Random Forest

📈 **Best Performance:**

* Accuracy: **92%**
* ROC-AUC: **0.92**

Hyperparameter tuning applied to improve model stability and performance.

### 4️⃣ Dimensionality Reduction

* Principal Component Analysis (PCA)
* Retained **86% of total variance**

### 5️⃣ Unsupervised Learning

* K-Means Clustering
* Hierarchical Clustering
* Identification of vehicle risk segments
